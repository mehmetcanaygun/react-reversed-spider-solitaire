## 112. Trendyol Front-End Bootcamp - Final Project: Reversed Spider Solitaire

## Subject

Create a Reversed Spider Solitaire game that users can line cards from "Ace" to "King".

## To run the project

```
npm install
```

```
npm start
```

```
npm run build
```
